# skull-stripper
Segmentation of mouse brain fMRI

## Depedencies
* Python 3.6.8
* Torch 1.3.1
* Torchvision 0.4.2
* Tensorboard 1.14.0
* TensorboardX 1.4
* Numpy 1.17.2
* Matplotlib 3.1.1
* Pillow 6.2.1
* scikit-image 0.16.2
* scikit-learn 0.21.3
